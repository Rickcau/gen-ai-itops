# Root Level Configuration Files:
1. components.json
 - Configuration for Shadcn UI components
 - Defines styling preferences, paths, and aliases
- Used by Shadcn CLI when adding new components

2. next-env.d.ts
- TypeScript declarations for Next.js
- Automatically generated by Next.js
- Should not be edited manually

3. package-lock.json
- Exact dependency tree for npm
- Ensures consistent installations across machines
- Generated automatically by npm

4. postcss.config.js
- PostCSS configuration
- Used by Tailwind CSS
- Handles CSS transformations and optimizations

5. tailwind.config.js
- Tailwind CSS configuration
- Defines custom colors, themes, and components
- Controls which files Tailwind should process

6. tsconfig.json
- TypeScript configuration
- Defines compiler options
- Sets up path aliases (like @/components)

## Additional Folders:
1. types/
- Custom TypeScript type definitions
- Contains interfaces like ChatApiRequest, Message, etc.
- Shared types used across components

2. .next/
- Build output directory
- Generated by Next.js
- Should be in .gitignore
